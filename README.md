
# Saboteur 矮人礦坑

# Intro your game
- 遊戲人數: 3~10 人
- 遊戲類型：	陣營、心機、派對
- 矮人礦坑(Saboteur)桌遊中，玩家會被分為兩個陣營，好矮人和壞矮人。玩家每回合必須出一張牌來連接道路、陷害或幫助別人、或是直接棄牌不出，然後再抽一張牌來補充自己的手牌。矮人礦坑的好矮人目標是將道路接到有金礦的目標牌上，壞矮人則是阻止好矮人達到目的。
